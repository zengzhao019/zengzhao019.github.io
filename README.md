# Zengzhao Li — Academic Pages Conversion

This repository is a fully converted Jekyll site inspired by Academic Pages while preserving the visual style of the previous custom homepage.

## What is included
- Modular Jekyll structure
- Home page + About + Research + Highlights + Publications + Contact
- Publication collection in `_publications/`
- Centralized CSS in `assets/css/main.css`
- Footer credit: `Powered by Jekyll & AcademicPages`

## Before publishing
1. Put your portrait image at `images/photo.jpg`
2. Put your CV at `files/cv.pdf`
3. Commit all files to `zengzhao019.github.io`
4. In GitHub Pages settings, build from the default branch root

## Run locally
```bash
bundle install
bundle exec jekyll serve
```
Then open `http://127.0.0.1:4000`
